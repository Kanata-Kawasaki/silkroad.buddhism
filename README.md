# シルクロード仏教文化研究会 ウェブサイト

このプロジェクトは、シルクロード仏教文化研究会のウェブサイトのソースコードです。

## セットアップ手順

1. このプロジェクトをクローンまたはダウンロードします。
2. 下記のファイル構造に従って、必要なフォルダを作成してください。
3. 背景画像として使用する画像を `images` フォルダに配置してください。
4. `index.html` をブラウザで開いてウェブサイトを表示します。

## ファイル構造

```
/
├── index.html          # メインのHTMLファイル
├── styles.css          # スタイルシート
├── script.js           # JavaScriptファイル
└── images/             # 画像フォルダ
    ├── bg-1.jpg        # 背景画像1（シルクロードの風景など）
    ├── bg-2.jpg        # 背景画像2（仏教遺跡など）
    ├── bg-3.jpg        # 背景画像3（石窟寺院の壁画など）
    ├── bg-4.jpg        # 背景画像4（中央アジアの風景など）
    ├── taniguchi.jpg   # 谷口陽子教授の写真
    └── hiyama.jpg      # 檜山智美助教の写真
```

## 背景画像について

ウェブサイトには以下の画像が必要です：

1. **フルスクリーン背景スライダー用画像**：
   - `bg-1.jpg`, `bg-2.jpg`, `bg-3.jpg`, `bg-4.jpg`
   - 高解像度（最低1920x1080ピクセル）で、良質なものを使用
   - これらは8秒ごとに自動的に切り替わり、拡大・縮小効果と滑らかな遷移効果が適用される
   - 暗めの画像が理想的（明るすぎると文字が見えづらくなる）

2. **セクション背景画像**：
   - 各セクション（研究会について、イベント、主催者、お問い合わせ）には、スライダーと同じ画像が使用される
   - パララックス効果が適用されるため、スクロールすると背景画像がゆっくりと動く

3. **主催者の写真**：
   - `taniguchi.jpg`：谷口陽子教授の写真
   - `hiyama.jpg`：檜山智美助教の写真

推奨される画像の内容:
- シルクロードの風景
- 仏教遺跡
- 石窟寺院の壁画
- 中央アジアの遺跡や風景

## 最新の機能と強化点

### 背景スライダーの強化
- **フルスクリーンカバー表示**：背景画像が画面全体を覆い、余白なく表示
- **スムーズな遷移アニメーション**：スライド間の切り替えがより滑らかに
- **スケールとスライド効果**：スライドイン時に拡大・縮小効果を追加
- **マウス追従効果**：マウスの動きに合わせて背景が微妙に動く視差効果
- **画像プリロード**：画像を先読みして切り替え時のチラつきを防止

### セクション背景の強化
- **パララックスエフェクト**：スクロールすると背景画像がゆっくりと動く効果
- **背景の視認性向上**：背景画像の表示方法を改善し、コンテンツの視認性を確保
- **マウス連動効果**：マウスの動きに連動して背景が微妙に動く
- **セクション別の色調変更**：各セクションごとに異なる色調フィルターを適用

### デザイン要素の強化
- **セクション区切りの装飾**：セクション間の境界を美しいパターンで装飾
- **グラスモーフィズム効果**：ヒーローセクションのコンテンツ背景にぼかし効果を適用
- **動的なグラデーション**：ヒーローセクションの背景に動くグラデーションを追加
- **テキスト輝き効果**：メインタイトルに輝くアニメーション効果を適用
- **立体的なボタンデザイン**：ボタンにグラデーションとホバーエフェクトを追加

## カスタマイズ

### 色の変更

`styles.css` ファイルの冒頭にある `:root` セレクタ内の変数を編集することで、
ウェブサイト全体の色調を簡単に変更できます。

```css
:root {
    --primary-color: #2c3e91;      /* 深い青 - メインカラー */
    --secondary-color: #0091ea;    /* 明るい青 - アクセントカラー */
    --accent-color: #689f38;       /* 緑色 - 補助カラー */
    --dark-blue: #0d1b42;          /* 暗い青 - フッター背景 */
    --light-blue: #cce5ff;         /* 薄い青 - セクション背景 */
    --text-color: #222;            /* 黒に近い色 - 基本テキスト */
    --text-light: #ffffff;         /* 白に近い色 - 明るいテキスト */
}
```

### 背景画像の変更

背景画像を変更するには、新しい画像を `images` フォルダに配置し、
`styles.css` ファイル内の以下の部分を編集してください。

```css
.bg-slider-item:nth-child(1) {
    background-image: url('images/bg-1.jpg');
}

.bg-slider-item:nth-child(2) {
    background-image: url('images/bg-2.jpg');
}

.bg-slider-item:nth-child(3) {
    background-image: url('images/bg-3.jpg');
}

.bg-slider-item:nth-child(4) {
    background-image: url('images/bg-4.jpg');
}
```

セクション背景画像も同様に変更できます：

```css
.about-bg-image {
    background-image: url('images/bg-1.jpg');
}

.events-bg-image {
    background-image: url('images/bg-2.jpg');
}

.organizers-bg-image {
    background-image: url('images/bg-3.jpg');
}

.contact-bg-image {
    background-image: url('images/bg-4.jpg');
}
```

### アニメーション速度の調整

スライダーのアニメーション速度を変更するには、`script.js` ファイル内の以下の部分を編集してください：

```javascript
// スライドを切り替える間隔（ミリ秒）
setInterval(changeSlide, 8000);
```

## 機能一覧

- **全画面背景スライダー**：高品質な背景画像が自動的に切り替わる
- **スライドアニメーション**：背景画像の切り替え時に滑らかな遷移効果
- **ズームエフェクト**：背景画像がゆっくりとズームイン・アウトするアニメーション
- **マウス連動パララックス**：マウスの動きに合わせて背景が動く視差効果
- **スクロールパララックス**：スクロール時に背景画像が視差効果で動く
- **セクション装飾**：セクション間の境界を美しいパターンで装飾
- **グラスモーフィズム**：ヒーローセクションに半透明のガラス調デザイン
- **レスポンシブデザイン**：あらゆる画面サイズに最適化されたレイアウト
- **スムーズスクロール**：ナビゲーションリンクのクリックで対応セクションにスムーズに移動
- **スクロールアニメーション**：セクションが画面内に入ると自動的にフェードインするアニメーション
- **グラデーションアニメーション**：ヒーローセクションの背景とボタンに動くグラデーション

## 注意点

- このウェブサイトはモダンなCSSとJavaScript機能を使用しており、最新のブラウザ（Chrome、Firefox、Safari、Edgeの最新版）での表示を推奨します。
- 背景画像は高解像度（最低1920x1080ピクセル）のものを使用してください。
- 画像が大きすぎると読み込み時間が長くなる可能性があります。必要に応じて最適化してください。
- 画像は著作権に配慮して使用してください。
- モバイルデバイスでは一部の視覚効果（マウスパララックスなど）が省略されます。
